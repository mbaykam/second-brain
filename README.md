## Mobile Sync (Free Options)

### Option 1: GitHub + Obsidian Git (Recommended)
1. Create free GitHub repo
2. Install "Obsidian Git" plugin
3. Configure auto-sync every 5 minutes
4. On mobile: Same setup, works seamlessly

### Option 2: Syncthing (Fully P2P)
1. Install Syncthing on desktop + phone
2. Share vault folder
3. Syncs directly device-to-device, no cloud

### Option 3: Self-hosted LiveSync
1. Set up free CouchDB (IBM Cloud, etc.)
2. Install "Self-hosted LiveSync" plugin
3. Real-time sync across devices

See `mobile-sync.md` for detailed setup.
